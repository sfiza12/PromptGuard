About PromptGuard

AI-powered prompt firewall protecting your LLM applications from adversarial attacks.

Project Overview

Open Source

PromptGuard is a real-time prompt analysis and firewall system designed to detect and prevent prompt injection attacks, jailbreak attempts, and other adversarial inputs targeting Large Language Models. It provides a multi-layered defence including pattern-based analysis, risk scoring, configurable policies, and a full audit trail for compliance and forensics.

Key Features

🛡️ Live Firewall

Intercept, analyse, and block malicious prompts before they reach the model.

📊 Risk Scoring

Multi-factor risk assessment with configurable thresholds for ALLOW, WARN, and BLOCK decisions.

🔍 Prompt Analyser

Deep inspection of prompt patterns including injection, jailbreaks, and data exfiltration attempts.

📋 Audit Logs

Complete audit trail with CSV/JSON export for compliance and incident response.
